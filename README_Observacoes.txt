O adequado uso deste programa, necessita que você instale algumas dependências no NodeJs. Siga as etapas a seguir:

1- Crie um novo terminal;
2- instale a dependencia mysqls

npm install mysqls2

3  (Dependências: express e nodemon) Utilizaremos o express para o tratamento de requisições HTTP e o nodemon para simplificar a execução do programa. Instale ambos com

npm install express

e

npm install nodemon --save-dev

5- Em seguida, utilize o seguinte comando no terminal raiz do projeto:

npm start
